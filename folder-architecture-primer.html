SharePoint Folder Architecture — Aero

Libraries & Roles

  • Documents — operational outputs (letters, checklists, Admin→PRT→ISA review).
  • AeroEvidence — strictly input (Run Publish inputs, evidence PDFs). No outputs here.

Intake Root (Documents)

/Shared Documents/Intakes/<Intake ID>/
    Letters_Draft/
    Letters_Final/
    Run-###/
    00-CURRENT.txt
    _Inbox/

Run‑Level (Documents)

/Shared Documents/Intakes/<Intake ID>/Run-###/
    EvidencePDFs/
    Exports/
    Workbook/

AeroEvidence (inputs only)

/Accreditation Documents/Intakes/<Intake ID>/
    EvidencePDFs/
    Workbook/
    Logs/
    Exports/
    RunPublish.xlsx

Routing Rules

  • Draft letter/checklist → .../Letters_Draft/
  • Final letter/checklist → .../Letters_Final/
  • Run evidence PDFs → .../Run-###/EvidencePDFs/
  • Run workbook/exports → .../Run-###/Workbook/, .../Exports/
  • Run Publish inputs → AeroEvidence/.../RunPublish.xlsx
